# Star-Score Stability Index (SSSI) — R Implementation

### Quantifying the Ecological-Economic Nexus
SSSI is a digital framework designed to challenge urban-centric bias in digital policies. It utilises indigenous knowledge as a primary driver for algorithmic accountability.

## Installation
 `tidyverse` and `ggiraphExtra` installed for the radar visualisations.

## How it Works
The index calculates a **Weighted Geometric Mean** between Environmental Health (E) and Household Productivity (P). By using a geometric mean, the score collapses if the environment fails, preventing "artificial success" masking.
